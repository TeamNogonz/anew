다음은 여러 언론사에서 수집한 뉴스 기사 리스트입니다. 각 기사는 언론사 이름(press), 뉴스 링크(url), 제목(title), 본문(content)으로 구성되어 있습니다.

작업을 다음과 같이 수행하세요:

1. 기사들을 분석하여 같은 주제를 다룬 기사들을 그룹화(클러스터링)하세요.
2. 그룹화된 주제들 중에서 가장 많이 다뤄진 주제를 정확히 {summary_news_count}개만 선정하세요.
3. 선정된 {summary_news_count}개의 주제에 대해서만 각각 서로 다른 관점 2가지로 요약하세요.
   - 관점은 예를 들어 "정부 관점 vs 시민 관점", "긍정적 관점 vs 부정적 관점" 등으로 다양할 수 있으며, 각 관점은 명확하게 구분되어야 합니다.
4. 각 관점의 요약은 최대 {max_length}자 이내로 작성하세요.
5. 각 주제별로 관련된 뉴스 기사들의 URL을 배열로 함께 제공하세요.

중요: 정확히 {summary_news_count}개의 주제만 선정하여 요약하세요. 모든 주제를 요약하지 마세요.

결과는 아래 JSON 형식으로 반환하세요:

[
    {
        "title": "<주제 제목>",
        "first_perspective": "<관점 1 기반 요약, 최대 {max_length}자>",
        "second_perspective": "<관점 2 기반 요약, 최대 {max_length}자>",
        "reference_url": ["<url1>", "<url2>", ...]
    },
    ...
]

중요: JSON 코드 블록 마커(```json, ```)를 사용하지 말고 순수한 JSON 객체만 반환하세요.

아래는 입력 뉴스 데이터입니다:
{news_json}
